# CreateCalendar

Google スプレッドシートの年間行事予定を、Google カレンダーへまとめて反映する Google Apps Script（GAS）です。  
祝日の追加・削除、毎週予定の一括追加・削除、全期間・前期・後期・月別のカレンダー書き込みに対応しています。

## 1. はじめに

まずは以下のテンプレート用スプレッドシートをコピーして使ってください。  
参照用 URL: https://docs.google.com/spreadsheets/d/1_qxd3jlTZU1qURFa5kEOxC1fZVR-Fe5Hn5Ytz1SPQRc/copy

## 2. 事前準備

1. コピーしたスプレッドシートを開きます。
2. `A1` に対象年度を入力します。例: `2026`
3. `E1` に書き込み先の Google カレンダー ID を入力します。  
   Google カレンダーの「設定と共有」→「カレンダーの統合」→「カレンダー ID」から取得できます。
4. スプレッドシートを再読み込みし、メニューに `年間行事予定` が表示されることを確認します。

## 3. 行事予定の入力ルール

- 終日予定: `入学式`
- 時間指定あり: `会議<10:00-12:00>`
- 複数予定: `授業参観,職員会議`
- 混在: `遠足<9:00-15:00>,職員研修`

全角英数字や全角記号が混ざっていても、ある程度は自動で正規化されます。  
ただし、時刻は `開始-終了` の形で入力してください。

## 4. メニューの使い方

`年間行事予定` メニューから次の機能を使えます。

- `【祝日追加】祝日を行事予定に追加`
- `【祝日削除】祝日を行事予定から削除`
- `【毎週追加】毎週の予定を追加`
- `【毎週削除】毎週の予定を削除`
- `【全期間登録】全期間カレンダーへ書き込み実行`
- `【前期登録】前期（4-9月）書き込み実行`
- `【後期登録】後期（10-3月）書き込み実行`
- `【月別登録】特定の月だけ書き込み実行`

## 5. 書き込み時の注意

- カレンダーへの書き込み実行前に、対象期間の既存予定を削除してから再登録します。
- 本番カレンダーでいきなり実行せず、最初はテスト用カレンダーで確認してください。
- カレンダー ID が誤っている、または権限が不足している場合は処理を中止します。
- 夜間跨ぎの予定（例: `宿直<23:00-01:00>`）にも対応しています。

## 6. ファイル構成

- `コード.js` : メニュー処理、予定登録、祝日・毎週予定の操作ロジック
- `appsscript.json` : GAS マニフェスト

## 7. よくあるエラーと対処法

### `Cannot read properties of null (reading 'createAllDayEvent')`

- 原因: `E1` のカレンダー ID から対象カレンダーを取得できていません。
- 対処: `E1` に Google カレンダーの「カレンダー ID」を正しく入力してください。
- あわせて、実行アカウントにそのカレンダーの編集権限があるか確認してください。

### `カレンダーIDで対象カレンダーを取得できませんでした`

- 原因: カレンダー ID の誤り、不要な空白、または権限不足が考えられます。
- 対処: `E1` の値を入れ直し、コピー元ではなく自分が編集できるカレンダーを指定してください。

### `スプレッドシートに有効な日付が見つかりません`

- 原因: カレンダー表の日付セルが日付として認識されていません。
- 対処: `A3:X33` の日付列に、文字列ではなく日付形式の値が入っているか確認してください。

### 時間指定の予定が登録されない

- 原因: 時刻の書式が想定外の可能性があります。
- 対処: `会議<10:00-12:00>` のように、`予定名<開始-終了>` の形式で入力してください。
- 複数予定を入れる場合は `授業参観,職員会議` のようにカンマ区切りで入力してください。

### 既存予定が消えてしまう

- 原因: 書き込みメニューは、対象期間の既存予定を削除してから再登録する仕様です。
- 対処: 初回は必ずテスト用カレンダーで実行し、本番カレンダーで使う前に挙動を確認してください。
